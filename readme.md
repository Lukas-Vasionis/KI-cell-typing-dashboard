# Installation
1. Clone this repository
2. In the project's directory (Terminal): Install dependencies by executing `pip freeze > requirements.txt` 
3. In the project's directory (Terminal): Execute the app `streamlit run app.py`
4. The app should open in your browser. Other wise, click on the link in your terminal. It should look something like this:
```
(base) user@user:KI-cell-typing-dashboard$ streamlit run app.py

  You can now view your Streamlit app in your browser.

  Local URL: http://localhost:1234
  Network URL: http://123.123.1.123:1234
```
