# Installation
1. Clone this repository
2. Move to project's directory and execute these in a terminal:
2. Install dependencies by executing `pip install -r ./requirements.txt` 
3. Execute the app `streamlit run app.py`

The app should open in your browser. Otherwise, click on the link in your terminal. It should look something like this:
```
(base) user@user:KI-cell-typing-dashboard$ streamlit run app.py

  You can now view your Streamlit app in your browser.

  Local URL: http://localhost:1234
  Network URL: http://123.123.1.123:1234
```
